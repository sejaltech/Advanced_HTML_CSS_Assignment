Navigation with ARIA

This navigation structure uses proper ARIA roles to ensure accessibility for screen reader users.

Skip to main content

Accessibility Features Used:

  • role="navigation": Identifies navigation landmarks
  • role="menu": Identifies dropdown menus
  • aria-label: Provides accessible names for elements
  • aria-haspopup: Indicates when an element has a submenu
  • aria-expanded: Shows if a menu is open or closed
  • aria-current="page": Marks the current page in breadcrumbs
  • aria-controls: Links toggle button to the menu it controls
